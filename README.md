# CGM

## How to install

```commandline
python setup.py install develop --user
```

## Training

```commandline
python scripts/train.py
```

## Evaluation

1. 15/30/60 min prediction
    ```commandline
    python scripts/test.py
    ```

2. Generate customized BG response
    ```commandline
    python scripts/test_customized_response.py
    ```
